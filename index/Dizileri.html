Dizi Oluşturma

Diziler, her biri için ayrı değişkenler bildirmek yerine, birden çok değeri tek bir değişkende depolamak için kullanılır değer.

Bir diziyi bildirmek için, değişken türünü köşeli parantez içinde tanımlayın:

string[] cars;

Şimdi bir dizi dize tutan bir değişken bildirdik.

Buna değer eklemek için, değişmez bir dizi kullanabiliriz - değerleri virgülle ayrılmış bir listeye, kıvırcık parantezlerin
içine yerleştirin:

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

Bir tamsayı dizisi oluşturmak için şunları yazabilirsiniz:

int[] myNum = {10, 20, 30, 40};

Bir Dizinin Öğelerine Erişme

Bir dizi öğesine dizin numarasına başvurarak erişirsiniz.

Bu ifade, otomobillerdeki ilk öğenin değerine erişir:

Örnek

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
Console.WriteLine(cars[0]); // Outputs Volvo

Not: Dizi dizinleri 0 ile başlar: [0] ilk öğedir. [1] ikincisidir elemanı, vb.

Dizi Öğesini Değiştirme

Belirli bir öğenin değerini değiştirmek için, dizin numarasına bakın:

Örnek

cars[0] = "Opel"

Örnek

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
cars[0] = "Opel";
Console.WriteLine(cars[0]);
// Now outputs Opel instead of Volvo

Dizi Uzunluğu

Bir dizinin kaç öğeye sahip olduğunu öğrenmek için şu özelliği kullanın: Length

Örnek

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
Console.WriteLine(cars.Length);
// Outputs 4

Dizi Oluşturmanın Diğer Yolları

C# hakkında bilginiz varsa, anahtar sözcükle oluşturulan dizileri görmüş olabilirsiniz ve belki de belirli bir boyuta sahip
diziler de görmüşsünüzdür. C#'ta, dizi oluşturmanın farklı yolları vardır:new

Örnek

// Create an array of four elements, and add values later
string[] cars = new string[4];

// Create an array of four elements and add values right away
string[] cars = new string[4] {"Volvo", "BMW", "Ford", "Mazda"};

// Create an array of four elements without specifying the size
string[] cars = new string[] {"Volvo", "BMW", "Ford", "Mazda"};

// Create an array of four elements, omitting the new keyword, and without specifying the size
string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

Hangi seçeneği seçeceğiniz size kalmış. Öğreticimizde, okunması daha hızlı ve daha kolay olduğu için genellikle son
seçeneği kullanacağız.

Ancak, bir diziyi bildirir ve daha sonra başlatırsanız anahtar sözcüğü kullanmanız gerektiğini unutmayın:new

Örnek

// Declare an array string[] cars;

// Add values, using new
cars = new string[] {"Volvo", "BMW", "Ford"};

// Add values without using new (this will cause an error)
cars = {"Volvo", "BMW", "Ford"};
