C# Anahtar İfadeleri

Yürütülecek birçok kod bloğundan birini seçmek için deyimi kullanın.switch

Örnek

switch(expression)
{
case x:
// code block
break;
case y:
// code block
break;
default:
// code block
break;
}

Süreç şu şekilde işler:

  • İfade bir kez değerlendirilirswitch
  • İfadenin değeri, her birinin değerleriyle karşılaştırılır case
  • Bir eşleşme varsa, ilişkili kod bloğu yürütülür
  • Ve anahtar kelimeler bu bölümde daha sonra açıklanacaktır.breakdefault

Aşağıdaki örnek, hafta içi gün adını hesaplamak için hafta içi numarasını kullanır:

Örnek

int day = 4;
switch (day)
{
case 1:
Console.WriteLine("Monday");
break;
case 2:
Console.WriteLine("Tuesday");
break;
case 3:
Console.WriteLine("Wednesday");
break;
case 4:
Console.WriteLine("Thursday");
break;
case 5:
Console.WriteLine("Friday");
break;
case 6:
Console.WriteLine("Saturday");
break;
case 7:
Console.WriteLine("Sunday");
break;
}
// Outputs "Thursday" (day 4)

break Anahtar Kelimesi

C# bir anahtar sözcüğe ulaştığında, anahtar bloğundan çıkar.break

Bu, içeride daha fazla kod ve servis talebi testinin yürütülmesini durduracaktır blok.

Bir eşleşme bulunduğunda ve iş bittiğinde, mola zamanı gelmiştir. Daha fazla teste gerek yoktur.

Bir mola, yürütmeyi "görmezden geldiği" için çok fazla yürütme süresi kazandırabilir anahtar bloğundaki kodun geri kalanının tümünden.

Varsayılan Anahtar Kelime

Anahtar sözcük isteğe bağlıdır ve aksi takdirde çalıştırılacak bazı kodları belirtir vaka eşleşmesi:default

Örnek

int day = 4;
switch (day)
{
case 6:
Console.WriteLine("Today is Saturday.");
break;
case 7:
Console.WriteLine("Today is Sunday.");
break;
default:
Console.WriteLine("Looking forward to the Weekend.");
break;
}
// Outputs "Looking forward to the Weekend."
