C# Booleans

Çoğu zaman, programlamada, aşağıdakiler gibi iki değerden yalnızca birine sahip olabilen bir veri türüne ihtiyacınız
olacaktır:

  • EVET / HAYIR
  • AÇIK / KAPALI
  • DOĞRU / YANLIŞ

Bunun için C#, değerleri veya .bool/true/false

Boole Değerleri

Bir boole türü anahtar kelimeyle bildirilir ve yalnızca değerleri veya :booltruefalse

Örnek

bool isCSharpFun = true;
bool isFishTasty = false;
Console.WriteLine(isCSharpFun); // Outputs True
Console.WriteLine(isFishTasty); // Outputs False

Ancak, koşullu sınama için boole ifadelerinden boole değerleri döndürmek daha yaygındır (aşağıya bakınız).

Boole İfadesi

Boole ifadesi bir boole değeri döndürür: veya , değerleri/değişkenleri karşılaştırarak.TrueFalse

Bu, mantık oluşturmak ve cevapları bulmak için kullanışlıdır.

Örneğin, bir ifadenin (veya değişkenin) doğru olup olmadığını öğrenmek için büyüktür () işleci gibi bir karşılaştırma işleci kullanabilirsiniz:>

Örnek

int x = 10;
int y = 9;
Console.WriteLine(x > y); // returns True, because 10 is higher than 9

Veya daha da kolay:

Örnek

Console.WriteLine(10 > 9); // returns True, because 10 is higher than 9

Aşağıdaki örneklerde, bir ifadeyi değerlendirmek için eşittir () işlecini kullanırız:==

Örnek

int x = 10;
Console.WriteLine(x == 10); // returns True, because the value of x is equal to 10

Örnek

Console.WriteLine(10 == 15); // returns False, because 10 is not equal to 15

Gerçek Hayat Örneği

Bir "gerçek" düşünelim yaşam örneği", bir kişinin oy kullanacak kadar yaşlı olup olmadığını öğrenmemiz gereken yer

Aşağıdaki örnekte, yaşın () yaşın OR ayarlı oylama yaş sınırına eşit olup olmadığını öğrenmek için karşılaştırma operatörünü kullanıyoruz. Hedef:>=2518

Örnek

int myAge = 25;
int votingAge = 18;
Console.WriteLine(myAge >= votingAge);

Harika, değil mi? Daha da iyi bir yaklaşım (şu anda bir ruloda olduğumuz için), yukarıdaki kodu bir ifadeye sarmak
olacaktır, böylece gerçekleştirebiliriz. sonuca bağlı olarak farklı eylemler:if...else

Örnek

int myAge = 25;
int votingAge = 18;

if (myAge >= votingAge)
{
Console.WriteLine("Old enough to vote!");
}
else
{
Console.WriteLine("Not old enough to vote."); }

Bir ifadenin boole değeri, tüm C# karşılaştırmalarının ve koşullarının temelidir.

Koşullar hakkında daha fazla bilgi edineceksiniz (eğer... else) bir sonraki bölümde.
