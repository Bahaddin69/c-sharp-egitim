C# Sonu

Bu öğreticinin önceki bir bölümünde kullanılan ifadeyi zaten gördünüz. Bir ifadeden "atlamak" için kullanıldı.breakswitch

İfade, bir döngüden atlamak için de kullanılabilir.break

Bu örnek, aşağıdakine eşit olduğunda döngüden atlar:i4

Örnek

for (int i = 0; i < 10; i++)
{
if (i == 4)
{
break;
}
Console.WriteLine(i);
}

C# Devam Et

İfade, belirtilen bir koşul oluşursa bir yinelemeyi (döngüde) keser ve döngüdeki bir sonraki yinelemeyle devam eder.
continue

Bu örnek şu değeri atlar:4

Örnek

for (int i = 0; i < 10; i++)
{
if (i == 4)
{
continue;
}
Console.WriteLine(i);
}

Döngü Sırasında Kırılın ve Devam Edin

Ayrıca ve while döngülerini de kullanabilirsiniz:breakcontinue

Ara Örneği

int i = 0; while (i < 10)
{
Console.WriteLine(i);
i++;
if (i == 4)
{
break;
}
}

Devam Et Örneği

int i = 0;
while (i < 10)
{
if (i == 4)
{
i++;
continue;
}
Console.WriteLine(i);
i++;
}
