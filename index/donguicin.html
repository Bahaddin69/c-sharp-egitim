Döngü için C#

Bir bloktan kaç kez geçmek istediğinizi tam olarak bildiğinizde kodu, döngü yerine döngüyü kullanın:for/while

Sözdizimi

for (statement 1; statement 2; statement 3)
{
// code block to be executed
}

Deyim 1, kod bloğunun yürütülmesinden önce (bir kez) yürütülür.

Deyim 2, kod bloğunu yürütmek için koşulu tanımlar.

Deyim 3, kod bloğu yürütüldükten sonra (her seferinde) yürütülür.

Aşağıdaki örnekte 0 ile 4 arasındaki sayılar yazdırılacaktır:

Örnek

for (int i = 0; i < 5; i++)
{
Console.WriteLine(i);
}

Örnek açıklandı

Deyim 1, döngü başlamadan önce bir değişken ayarlar ().int i = 0

Deyim 2, döngünün çalışması için koşulu tanımlar ( 'den küçük olmalıdır). Eğer koşul , ise döngü yeniden başlayacaktır,
eğer öyleyse, döngü sona erecektir.i5truefalse

Deyim 3, döngüdeki kod bloğunun sahip olduğu her seferinde bir değer () artırır idam edildi.i++

Başka Bir Örnek

Bu örnek yalnızca 0 ile 10 arasındaki çift değerleri yazdırır:

Örnek

for (int i = 0; i <= 10; i=i + 2)
{
Console.WriteLine(i);
}

İç İçe Döngüler

Bir döngüyü başka bir döngünün içine yerleştirmek de mümkündür. Buna iç içe döngü denir.

"İç döngü", "dış döngünün" her yinelemesi için bir kez yürütülür:

Örnek

// Outer loop
for (int i = 1; i <= 2; ++i)
{
Console.WriteLine("Outer: " + i); // Executes 2 times

// Inner loop
for (int j = 1; j <= 3; j++)
{
Console.WriteLine(" Inner: " + j); // Executes 6 times (2 * 3)
}
}
