Döngü

Döngüler, belirtilen bir koşula ulaşıldığı sürece bir kod bloğunu yürütebilir.

Döngüler kullanışlıdır çünkü zamandan tasarruf sağlarlar, hataları azaltırlar ve kodu daha okunabilir hale getirirler.

C# Döngü Sırasında

Döngü, belirtilen bir koşul aşağıdaki gibi olduğu sürece bir kod bloğu boyunca döngü yapar:whileTrue

Sözdizimi

while (condition)
{
// code block to be executed
}

Aşağıdaki örnekte, döngüdeki kod, aşağıdaki süre boyunca tekrar tekrar çalışacaktır: bir değişken (i) 5'ten küçükse:

Örnek

int i = 0;
while (i < 5)
{
Console.WriteLine(i);
i++;
}

Not: Durumda kullanılan değişkeni arttırmayı unutmayın, aksi takdirde döngü asla bitmeyecek!

Do/While Döngüsü

Döngü, döngünün bir çeşididir. Bu döngü Koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez
yürütün, ardından koşul doğru olduğu sürece döngüyü yineleyin.do/whilewhile

Sözdizimi

do
{
// code block to be executed
}
while (condition);

Aşağıdaki örnekte bir döngü kullanılmıştır. Döngü her zaman şöyle olacaktır: koşul yanlış olsa bile en az bir kez yürütülür,
çünkü kod bloğu koşul sınanmadan önce yürütülür:do/while

Sözdizimi

int i = 0;
do
{
Console.WriteLine(i);
i++;
}
while (i < 5);

Durumda kullanılan değişkeni arttırmayı unutmayın, aksi takdirde döngü asla bitmeyecek!
