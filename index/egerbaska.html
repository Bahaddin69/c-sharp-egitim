C# Koşulları ve If İfadeleri

C#, matematikteki olağan mantıksal koşulları destekler:

  • Daha az: a < b
  • Küçüktür veya eşittir: a <= b
  • Şundan büyük: a > b
  • Büyük veya eşittir: a >= b
  • a'ya eşit == b
  • Eşit Değil: a != b

Farklı kararlar için farklı eylemler gerçekleştirmek üzere bu koşulları kullanabilirsiniz.

C# aşağıdaki koşullu ifadelere sahiptir:

  • Belirtilen bir koşul doğruysa, yürütülecek bir kod bloğunu belirtmek için kullanınif
  • Aynı koşul yanlışsa, yürütülecek bir kod bloğu belirtmek için kullanınelse
  • İlk koşul yanlışsa, test edilecek yeni bir koşul belirtmek için kullanınelse if
  • Yürütülecek birçok alternatif kod bloğunu belirtmek için kullanınswitch

if Beyanı

Bir koşul .if/True

Örnek

if (condition)
{
// block of code to be executed if the condition is True
}

Bunun küçük harflerle yazıldığını unutmayın. Büyük harfler (If veya IF) bir hata oluşturur.if

Aşağıdaki örnekte, 20'nin büyük olup olmadığını öğrenmek için iki değeri test ediyoruz. 18. Koşul ise,
bazı metinler yazdırın:True

Örnek

if (20 > 18)
{
Console.WriteLine("20 is greater than 18");
}

Ayrıca değişkenleri de test edebiliriz:

Örnek

int x = 20;
int y = 18;
if (x > y)
{
Console.WriteLine("x is greater than y");
}

Örnek açıklandı

Yukarıdaki örnekte x ve y olmak üzere iki değişken kullanıyoruz, x'in y'den büyük olup olmadığını test etmek için (operatörü kullanarak). X 20 ve y 18 olduğundan ve 20'nin 18'den büyük olduğunu bildiğimiz için, ekrana "x y'den büyüktür"
diye yazdırırız.>
