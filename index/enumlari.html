C# Enumları

An, bir grup sabiti (değiştirilemez/salt okunur değişkenler) temsil eden özel bir "sınıf"tır.enum

Bir anahtar sözcük oluşturmak için, anahtar sözcüğü kullanın (sınıf veya arabirim yerine) ve virgüllü enum
öğeleri:enum/enum

Örnek

enum Level
{
Low,
Medium,
High
}

Öğelere nokta sözdizimi ile erişebilirsiniz:enum

Level myVar = Level.Medium;
Console.WriteLine(myVar);

Enum, "özel olarak listelenmiş" anlamına gelen "numaralandırmalar" ın kısaltmasıdır.

Sınıf İçinde Enum

Ayrıca bir sınıfın içinde de olabilir:enum

Örnek

class Program
{
enum Level
{
Low,
Medium,
High
}
static void Main(string[] args)
{
Level myVar = Level.Medium;
Console.WriteLine(myVar);
}
}

Çıktı şöyle olacaktır:

Medium

Enum Değerleri

Varsayılan olarak, bir enumun ilk öğesi 0 değerine sahiptir. İkincisi 1 değerine sahiptir ve bu şekilde devam eder.

Bir öğeden tamsayı değerini almak için, öğeyi açıkça bir :int

Örnek

enum Months
{
January, // 0
February, // 1
March, // 2
April, // 3
May, // 4
June, // 5
July // 6
}

static void Main(string[] args)
{
int myNum = (int) Months.April;
Console.WriteLine(myNum);
}

Çıktı şöyle olacaktır:

3

Kendi enum değerlerinizi de atayabilirsiniz ve sonraki öğeler numaralarını buna göre güncelleyecektir:

Örnek

enum Months
{
January, // 0
February, // 1
March=6, // 6
April, // 7
May, // 8
June, // 9
July // 10
}

static void Main(string[] args)
{
int myNum = (int) Months.April;
Console.WriteLine(myNum);
}

Çıktı şöyle olacaktır:

7

Bir Switch İfadesinde Enum

Enumlar genellikle karşılık gelen değerleri denetlemek için ifadelerde kullanılır:switch

Örnek

enum Level
{
Low,
Medium,
High
}

static void Main(string[] args)
{
Level myVar = Level.Medium;
switch(myVar)
{
case Level.Low:
Console.WriteLine("Low level");
break;
case Level.Medium:
Console.WriteLine("Medium level");
break;
case Level.High:
Console.WriteLine("High level");
break;
}
}

Çıktı şöyle olacaktır:

Medium level

Ay günleri, günler, renkler, kart destesi gibi değişmeyeceğini bildiğiniz değerleriniz olduğunda enumları kullanın.
