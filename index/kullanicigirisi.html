Kullanıcı Girişi

Değerleri yazdırmak (yazdırmak) için kullanıldığını zaten öğrendiniz. Şimdi kullanıcı girdisi almak için kullanacağız.Console.WriteLine() Console.ReadLine()

Aşağıdaki örnekte, kullanıcı adını girebilir ve bu kullanıcı adı değişken. Sonra değerini yazdırıyoruz:userName userName

Örnek

// Type your username and press enter
Console.WriteLine("Enter username:");

// Create a string variable and get user input from the keyboard and store it in the variable
string userName = Console.ReadLine();

// Print the value of the variable (userName), which will display the input value
Console.WriteLine("Username is: " + userName);

Kullanıcı Girişi ve Sayıları

Yöntem bir . Bu nedenle, . Aşağıdaki program bir hataya neden olur:Console.ReadLine()stringint

Örnek

Console.WriteLine("Enter your age:");
int age = Console.ReadLine();
Console.WriteLine("Your age is: " + age);

Hata iletisi şöyle bir şey olacaktır:

Cannot implicitly convert type 'string' to 'int'

Hata mesajında belirtildiği gibi, 'string' türünü örtülü olarak 'int'

Neyse ki, sizin için, önceki bölümden (Type Casting) dönüştürebileceğinizi öğrendiniz. yöntemlerden birini kullanarak
herhangi bir türü açıkça kullanarak: Convert.To

Örnek

Console.WriteLine("Enter your age:");
int age = Convert.ToInt32(Console.ReadLine());
Console.WriteLine("Your age is: " + age);

Not: Yanlış giriş girerseniz (ör. sayısal girişteki metin), bir özel durum/hata iletisi alırsınız (System.FormatException: 'Giriş dizesi doğru biçimde değildi.').
Özel Durumlar ve hataların nasıl işleneceği hakkında daha sonraki bir bölümde daha fazla bilgi edineceksiniz.
