Kurucular

Oluşturucu, başlatmak için kullanılan özel bir yöntemdir Nesne. Bir yapıcının avantajı, bir sınıfın nesnesi oluşturulduğunda çağrılmasıdır. Şunları yapabilir: alanların başlangıç değerlerini ayarlamak için kullanılır:

Örnek

Bir oluşturucu oluşturun:

// Create a Car class class Car
{
public string model; // Create a field

// Create a class constructor for the Car class
public Car()
{
model = "Mustang"; // Set the initial value for model
}

static void Main(string[] args)
{
Car Ford = new Car(); // Create an object of the Car Class (this will call the constructor)
Console.WriteLine(Ford.model); // Print the value of model
}
}

// Outputs "Mustang"

Oluşturucu adının sınıf adıyla eşleşmesi gerektiğini ve bir dönüş türüne (like veya ) sahip olamayacağını unutmayın. void/int

Ayrıca, nesne oluşturulduğunda oluşturucunun çağrıldığını unutmayın.

Varsayılan olarak tüm sınıfların oluşturucuları vardır: bir sınıf oluşturmazsanız Kendiniz oluşturun, C# sizin için bir tane oluşturur. Ancak, o zaman yapamazsınız Alanlar için başlangıç değerlerini ayarlamak üzere ayarlayın.
İnşaatçılar zaman kazandırır! Nedenini gerçekten anlamak için bu sayfadaki son örneğe bir göz atın.

Oluşturucu Parametreleri

Oluşturucular, başlatmak için kullanılan parametreleri de alabilir Alanları.

Aşağıdaki örnek bir parametre ekler yapıcıya. Oluşturucunun içinde () olarak ayarladık. Aradığımızda Yapıcı,
oluşturucuya () değerini ayarlayacak bir parametre iletiriz:stringmodelName/model/modelName/model=modelName"Mustang"model"
Mustang"

Örnek

class Car
{
public string model;

// Create a class constructor with a parameter
public Car(string modelName)
{
model = modelName;
}

static void Main(string[] args)
{
Car Ford = new Car("Mustang");
Console.WriteLine(Ford.model);
}
}

// Outputs "Mustang"

İstediğiniz kadar parametreye sahip olabilirsiniz:

Örnek

class Car
{
public string model;
public string color;
public int year;

// Create a class constructor with multiple parameters
public Car(string modelName, string modelColor, int modelYear)
{
model = modelName;
color = modelColor;
year = modelYear;
}

static void Main(string[] args)
{
Car Ford = new Car("Mustang", "Red", 1969);
Console.WriteLine(Ford.color + " " + Ford.year + " " + Ford.model);
}
}


// Outputs Red 1969 Mustang

Bahşiş: Diğer yöntemlerde olduğu gibi, oluşturucular farklı sayıda parametre kullanılarak aşırı yüklenebilir.
