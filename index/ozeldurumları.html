C# Özel Durumları

C# kodunu çalıştırırken, farklı hatalar oluşabilir: programcı tarafından yapılan kodlama hataları, yanlış girişten
kaynaklanan hatalar, veya diğer öngörülemeyen şeyler.

Bir hata oluştuğunda, C# normalde durur ve bir hata iletisi oluşturur. Bunun için teknik terim şudur:
C# bir istisna atar (hata atar).

C# deneyin ve yakalayın

İfade, bir kod bloğu tanımlamanıza olanak tanır yürütülürken hatalar için test edilmiştir.try

İfade, bir kod bloğu tanımlamanıza olanak tanır: deneme bloğunda bir hata oluşursa yürütülür.catch

Ve anahtar kelimeler çiftler halinde gel:trycatch

Sözdizimi

try
{
// Block of code to try
}
catch (Exception e)
{
// Block of code to handle errors
}

Üç tamsayıdan oluşan bir dizi oluşturduğumuz aşağıdaki örneği ele alalım:

Bu bir hata oluşturur, çünkü myNumbers[10] mevcut değildir.

int[] myNumbers = {1, 2, 3};
Console.WriteLine(myNumbers[10]); // error!

Hata iletisi şöyle bir şey olacaktır:

System.IndexOutOfRangeException: 'Index was outside the bounds of the array.'

Bir hata oluşursa, hatayı yakalamak ve işlemek için bazı kodlar yürütmek için kullanabiliriz.try...catch

Aşağıdaki örnekte, yakalama bloğunun () içindeki değişkeni, özel durumu açıklayan bir ileti veren yerleşik özellik ile
birlikte kullanırız:e/Message

Örnek

try
{
int[] myNumbers = {1, 2, 3};
Console.WriteLine(myNumbers[10]);
}
catch (Exception e)
{
Console.WriteLine(e.Message);
}


Çıktı şöyle olacaktır:

Index was outside the bounds of the array.

Kendi hata mesajınızı da alabilirsiniz:

Örnek

try
{
int[] myNumbers = {1, 2, 3};
Console.WriteLine(myNumbers[10]);
}
catch (Exception e)
{
Console.WriteLine("Something went wrong.");
}

Çıktı şöyle olacaktır:

Something went wrong.

Nihayet

İfade, sonuçtan bağımsız olarak , sonra kod yürütmenizi sağlar:finally/try...catch

Örnek

try
{
int[] myNumbers = {1, 2, 3};
Console.WriteLine(myNumbers[10]);
}
catch (Exception e)
{
Console.WriteLine("Something went wrong.");
}
finally
{
Console.WriteLine("The 'try catch' is finished.");
}

Çıktı şöyle olacaktır:

Something went wrong.
The 'try catch' is finished.

Throw anahtar sözcüğü

İfade, özel bir hata oluşturmanıza olanak tanır.throw

İfade bir özel durum sınıfıyla birlikte kullanılır. C#'ta kullanılabilen birçok özel durum sınıfı vardır: , , , , , vb:throw ArithmeticException FileNotFoundException IndexOutOfRangeException TimeOutException

Örnek

static void checkAge(int age)
{
if (age < 18)
{
throw new ArithmeticException("Access denied - You must be at least 18 years old.");
}
else
{
Console.WriteLine("Access granted - You are old enough!");
}
}

static void Main(string[] args)
{
checkAge(15);
}

Programda görüntülenen hata mesajı şöyle olacaktır:

System.ArithmeticException: 'Access denied - You must be at least 18 years old.'

20 olsaydı, bir istisna elde etmezdiniz:age

Örnek

checkAge(20);

Çıktı şöyle olacaktır:

Access granted - You are old enough!
