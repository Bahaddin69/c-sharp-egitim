Sınıf Üyeleri

Sınıflar içindeki alanlar ve yöntemler genellikle "Sınıf Üyeleri" olarak adlandırılır:

Örnek

Üç sınıf üyesine sahip bir sınıf oluşturun: iki alan ve bir yöntem.Car

class Car
{
string color = "red";
}

Alanları

Önceki bölümde, bir sınıfın içindeki değişkenlerin alan olarak adlandırıldığını öğrendiniz, ve sınıfın bir nesnesini oluşturarak
ve nokta sözdizimini () kullanarak bunlara erişebileceğinizi unutmayın..

Aşağıdaki örnek, sınıfın bir nesnesini ad. Sonra değerini yazdırıyoruz alanları ve :Car/myObj/color/maxSpeed

Örnek

class Car
{
string color = "red";
int maxSpeed = 200;

static void Main(string[] args)
{
Car myObj = new Car();
Console.WriteLine(myObj.color);
Console.WriteLine(myObj.maxSpeed);
}
}

Ayrıca alanları boş bırakabilir ve nesneyi oluştururken bunları değiştirebilirsiniz:

Örnek

class Car
{
string color;
int maxSpeed;

static void Main(string[] args)
{
Car myObj = new Car();
myObj.color = "red";
myObj.maxSpeed = 200;
Console.WriteLine(myObj.color);
Console.WriteLine(myObj.maxSpeed);
}
}

Bu, özellikle bir sınıfın birden çok nesnesini oluştururken kullanışlıdır:

Örnek

class Car
{
string model;
string color;
int year;

static void Main(string[] args)
{
Car Ford = new Car();
Ford.model = "Mustang";
Ford.color = "red";
Ford.year = 1969;

Car Opel = new Car();
Opel.model = "Astra";
Opel.color = "white";
Opel.year = 2005;

Console.WriteLine(Ford.model);
Console.WriteLine(Opel.model);
}
}

Nesne Yöntemleri

C# Yöntemleri bölümünden, belirli eylemleri gerçekleştirmek için yöntemlerin kullanıldığını öğrendiniz.

Yöntemler normalde bir sınıfa aittir ve bir sınıfın nesnesinin nasıl davranacağını tanımlar.

Alanlarda olduğu gibi, nokta sözdizimi ile yöntemlere erişebilirsiniz. Ancak, şunu unutmayın: yöntemi olmalıdır.
Ve yöntemi çağırmak (yürütmek) için yöntemin adını ve ardından iki parantezin ve noktalı virgülün geldiğini
unutmayın:public();

Örnek

class Car
{
string color; // field
int maxSpeed; // field
public void fullThrottle() // method
{
Console.WriteLine("The car is going as fast as it can!");
}

static void Main(string[] args)
{
Car myObj = new Car();
myObj.fullThrottle(); // Call the method
}
}

Yöntemi neden C# Yöntemleri Bölümü'ndeki örneklerde olduğu gibi , olarak değil, , olarak ilan ettik?publicstatic

Nedeni basittir: Metne sınıfın bir nesnesi oluşturulmadan erişilebilirken, yöntemlere yalnızca nesneler tarafından erişilebilir.staticpublic

Birden Çok Sınıf Kullanma

Son bölümden, daha iyi organizasyon için birden fazla sınıf kullanabileceğimizi unutmayın. (biri alanlar ve yöntemler için,
diğeriyürütme için). Bu Önerilen:

prog2.cs

class Car
{
public string model;
public string color;
public int year;
public void fullThrottle()
{
Console.WriteLine("The car is going as fast as it can!");
}
}

prog.cs

class Program
{
static void Main(string[] args)
{
Car Ford = new Car();
Ford.model = "Mustang";
Ford.color = "red";
Ford.year = 1969;

Car Opel = new Car();
Opel.model = "Astra";
Opel.color = "white";
Opel.year = 2005;

Console.WriteLine(Ford.model);
Console.WriteLine(Opel.model);
}
}

Anahtar kelimeye erişim değiştirici denir, alanlarının diğer sınıflar için de erişilebilir olduğunu belirtir, örneğin, .public/Car Program
