C# Tipi Döküm

Tür dökümü, bir veri türünün değerini başka bir türe atadığınız zamandır.

C#'ta iki tür döküm vardır:

    Örtük Döküm (otomatik) - daha küçük bir türü dönüştürme daha büyük bir tür boyutuna
    char -> int -> long -> float -> double
  • Açık Döküm (manuel) - daha büyük bir türü dönüştürme daha küçük boyutlu bir türe
    double -> float -> long -> int -> char

Örtülü Döküm

Örtük döküm, daha küçük boyutlu bir tipin daha büyük boy tipi:

Örnek

double myDouble = 9.78;
int myInt = (int) myDouble; // Manual casting: double to int

Console.WriteLine(myDouble); // Outputs 9.78
Console.WriteLine(myInt); // Outputs 9

Açık Döküm

Açık döküm, türü parantez içine alarak manuel olarak yapılmalıdır değerin önünde:

Örnek

double myDouble = 9.78;
int myInt = (int) myDouble; // Manual casting: double to int

Console.WriteLine(myDouble); // Outputs 9.78
Console.WriteLine(myInt); // Outputs 9

Tür Dönüştürme Yöntemleri

Veri türlerini, , , , () ve () gibi yerleşik yöntemleri kullanarak açıkça dönüştürmek de mümkündür:Convert.ToBooleanConvert.ToDoubleConvert.ToStringConvert.ToInt32intConvert.ToInt64long

Örnek

int myInt = 10;
double myDouble = 5.25;
myBool = true;

Console.WriteLine(Convert.ToString(myInt)); // convert int to string
Console.WriteLine(Convert.ToDouble(myInt)); // convert int to double
Console.WriteLine(Convert.ToInt32(myDouble)); // convert double to int
Console.WriteLine(Convert.ToString(myBool)); // convert bool to string

Neden Dönüşüm?

Çoğu zaman, tür dönüştürmeye gerek yoktur. Ama bazen yapmak zorundasın. Bunun bir örneğini görmek için kullanıcı girişiyle çalışırken bir sonraki bölüme göz atın.
