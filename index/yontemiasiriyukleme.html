Yöntem Aşırı Yükleme

Yöntemin aşırı yüklenmesiyle, birden çok yöntem farklı adla aynı ada sahip olabilir Parametre:

Örnek

int MyMethod(int x)
float MyMethod(float x)
double MyMethod(double x, double y)

Farklı türde sayılar ekleyen iki yöntemi olan aşağıdaki örneği göz önünde bulundurun:

Örnek

static int PlusMethodInt(int x, int y)
{
return x + y;
}

static double PlusMethodDouble(double x, double y)
{
return x + y;
}

static void Main(string[] args)
{
int myNum1 = PlusMethodInt(8, 5);
double myNum2 = PlusMethodDouble(4.3, 6.26);
Console.WriteLine("Int: " + myNum1);
Console.WriteLine("Double: " + myNum2);
}

Aynı şeyi yapması gereken iki yöntem tanımlamak yerine, birini aşırı yüklemek daha iyidir.

Aşağıdaki örnekte, yöntemi her ikisi için de çalışacak şekilde aşırı yüklüyoruz ve :PlusMethod/int/double

Örnek

static int PlusMethod(int x, int y)
{
return x + y;
}

static double PlusMethod(double x, double y)
{
return x + y;
}

static void Main(string[] args)
{
int myNum1 = PlusMethod(8, 5);
double myNum2 = PlusMethod(4.3, 6.26);
Console.WriteLine("Int: " + myNum1);
Console.WriteLine("Double: " + myNum2);
}

Not: Birden çok yöntem aynı ada sahip olabilir parametrelerin sayısı ve/veya türü farklı olduğu sürece.
