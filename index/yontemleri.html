Yöntem, yalnızca çağrıldığında çalışan bir kod bloğudur.

Parametreler olarak bilinen verileri bir yönteme geçirebilirsiniz.

Yöntemler belirli eylemleri gerçekleştirmek için kullanılır ve bunlar işlevler olarak da bilinir.

Neden yöntemler kullanılmalı? Kodu yeniden kullanmak için: kodu bir kez tanımlayın ve birçok kez.

Yöntem Oluşturma

Bir yöntem tanımlanır yöntemin adıyla, ardından parantez (). C# gibi zaten aşina olduğunuz önceden tanımlanmış bazı
yöntemler sağlar, ancak belirli eylemleri gerçekleştirmek için kendi yöntemlerinizi de oluşturabilirsiniz:Main()

Örnek

Program sınıfı içinde bir yöntem oluşturun:Create a method within the Program class:

class Program
{
static void MyMethod()
{
// code to be executed
}
}

Örnek Açıklaması

  • MyMethod() yöntemin adıdır
  • static yöntemin Program sınıfının bir nesnesine değil, Program sınıfına ait olduğu anlamına gelir. Bu öğreticinin
    ilerleyen bölümlerinde nesneler ve nesneler aracılığıyla yöntemlere nasıl erişileceği hakkında daha fazla bilgi
    edineceksiniz.
  • void bu yöntemin bir dönüş değeri. Bu bölümün ilerleyen bölümlerinde dönüş değerleri hakkında daha fazla bilgi
    edineceksiniz
Not: C#'ta, kuralları adlandırırken büyük harfle başlamak iyi bir uygulamadır, çünkü kodun okunmasını kolaylaştırır.

Bir Yöntem Çağırma

Bir yöntemi çağırmak (yürütmek) için, yöntemin adını ve ardından iki tane yazın parantez () ve noktalı virgül;

Aşağıdaki örnekte, çağrıldığında bir metni (eylemi) yazdırmak için kullanılır:MyMethod()

Örnek

İçeride , yöntemi çağırın:Main() myMethod()

static void MyMethod()
{
Console.WriteLine("I just got executed!");
}

static void Main(string[] args)
{
MyMethod();
}

// Outputs "I just got executed!"

Bir yöntem birden çok kez çağrılabilir:

Örnek

static void MyMethod()
{
Console.WriteLine("I just got executed!");
}

static void Main(string[] args)
{
MyMethod();
MyMethod();
MyMethod();
}

// I just got executed!
// I just got executed!
// I just got executed!
