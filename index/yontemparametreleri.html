Parametreler ve Bağımsız Değişkenler

Bilgiler parametreler olarak yöntemlere aktarılabilir. Parametreler yöntemin içinde değişkenler gibi davranır.

Bunlar, yöntem adından sonra, parantez içinde belirtilir. İstediğiniz kadar parametre ekleyebilirsiniz, bunları virgülle
ayırmanız yeterlidir

Aşağıdaki örnekte parametre olarak adlandırılan bir fname'i alan yöntem. Yöntem çağrıldığında, bir ad veririz, tam adı
yazdırmak için yöntemin içinde kullanılan: string

Örnek

static void MyMethod(string fname)
{
Console.WriteLine(fname + " Refsnes");
}

static void Main(string[] args)
{
MyMethod("Liam");
MyMethod("Jenny");
MyMethod("Anja");
}

// Liam Refsnes
// Jenny Refsnes
// Anja Refsnes

Yönteme bir parametre geçirildiğinde, buna bağımsız değişken denir. Yani, yukarıdaki örnekten: bir parametredir, while , ve bağımsız değişkenlerdir.fnameLiamJennyAnja

Çoklu Parametreler

İstediğiniz kadar parametreye sahip olabilirsiniz, bunları virgülle ayırmanız yeterlidir:

Örnek

static void MyMethod(string fname, int age)
{
Console.WriteLine(fname + " is " + age);
}

static void Main(string[] args)
{
MyMethod("Liam", 5);
MyMethod("Jenny", 8);
MyMethod("Anja", 31);
}

// Liam is 5
// Jenny is 8
// Anja is 31

Birden çok parametreyle çalışırken, yöntem çağrısının parametrelerle aynı sayıda bağımsız değişkene sahip olmalıdır ve bağımsız değişkenler aynı sırada geçirilmelidir.
